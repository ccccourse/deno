# Tensor

This is a Tensor library (ndarray) for deno.js

## Import

```ts
import * as T from "https://ccc-js.github.io/tensor/mod.ts";
```

## Usage

```ts
import * as T from "https://ccc-js.github.io/tensor/mod.ts";

let a = [1,2,3], b = [1,1,1]
console.log('add(a,b) = ', T.add(a,b));

```
